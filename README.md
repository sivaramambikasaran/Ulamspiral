Ulam Spiral
===========

<img style = "float: center" src ="https://raw.githubusercontent.com/sivaramambikasaran/Ulamspiral/master/UlamSpiral_1.png" width="80%"/>

Generates different visualization of the Ulam spiral.

To obtain these visualization, do as follows: First run

	./runUlamSpiral.sh

Running this will ask you the following question:

	Enter the semi-length of the square: (a positive integer)

Key in some number say '40'. The next question you need to answer is:

	Enter the type of plot you want: (1, 2, 3)

	
Keying in '1' generate a .pdf file named "UlamSpiral_1.pdf"

<img style = "float: center" src ="https://raw.githubusercontent.com/sivaramambikasaran/Ulamspiral/master/UlamSpiral_1.png" width="80%"/>

Keying in '2' generate a .pdf file named "UlamSpiral_2.pdf"

<img style = "float: center" src ="https://raw.githubusercontent.com/sivaramambikasaran/Ulamspiral/master/UlamSpiral_2.png" width="80%"/>

Keying in semi-length = '150' and the option = '3' generates a .pdf file named "UlamSpiral_3.pdf"
